# WeChatTool_WhatToEat
还在对着冰箱里的菜发呆，想不起买了啥、不知道做什么饭？这款家庭菜品管理小程序，就是你的贴心饮食小助手。 随手拍张照片，就能自动识别菜品并记录，分类管理、标注保质期，再也不怕食材放坏浪费；根据冰箱里现有的菜，结合你想吃的健康清淡口，或是解馋的高热量美味，一键生成带详细步骤的食谱，还会标注卡路里、蛋白质等健康指数，吃饭不用再纠结；在这里，你能分享自己的做菜心得、美食日常，和同好们点赞评论、收藏心仪食谱；每周每月还会生成专属健康报告，贴心分析你的饮食均衡度，给出实用小建议。 从管好家里的菜，到吃好每一餐，让家常吃饭这件事，变得省心又健康。


后续需要配置的事项
云开发环境ID：在 miniprogram/config/env.js 中填写环境ID
tabBar图标：准备图标文件并放置到 miniprogram/images/tabbar/ 目录
AI API配置：在 miniprogram/config/api.js 中配置AI模型API（如需要）
云函数部署：上传并部署云函数到云开发环境
