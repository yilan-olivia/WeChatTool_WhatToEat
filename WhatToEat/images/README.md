# 图片资源目录

## 目录结构

```
images/
├── tabbar/          # tabBar图标
│   ├── home.png
│   ├── home-active.png
│   ├── food.png
│   ├── food-active.png
│   ├── community.png
│   ├── community-active.png
│   ├── profile.png
│   └── profile-active.png
├── default-avatar.png    # 默认头像
└── default-food.png      # 默认菜品图片
```

## 图标要求

- tabBar图标尺寸：81px × 81px
- 图标格式：PNG，支持透明背景
- 普通状态和选中状态需要提供两套图标

## 说明

请将实际的图标文件放置到对应目录中，替换占位文件。
